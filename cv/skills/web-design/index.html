Diseño web

De pequeño me movía mucho en foros, y creo que ese fue mi primer contacto con HTML. Más tarde, con mi primer blog, aprendí a estilizarlo y a partir de ahí he tenido varios proyectos relacionados con el diseño web.

Intento que está web sirva como ejemplo de mi conocimiento y mi filosofía en este ámbito. Aquí puedes ver la primera versión que hice de este sitio.

¿Qué sé hacer?

  • Sé crear páginas web atractivas con y sin JavaScript. Intento reducir al máximo el uso de JS en esta página para dar un ejemplo claro.
  • Puedo aplicar toda mi formación en Experiencia de Usuario para crear webs accesibles y atractivas sobre cualquier marca o iniciativa.
  • Manejo algunos diseñadores web como Wix y Framer, pero también sé crear páginas web adaptables desde cero, como esta.

Puedes consultar el código fuente de esta web en GitHub para ver cómo trabajo.
