Programación 🤖

La programación es una de mis mayores aficiones. Aprendo en mi tiempo libre y he realizado proyectos como un reproductor de música, un buscador de cervezas o incluso un pequeño sistema operativo llamado Blaze.

Gracias a todos estos proyectos tengo cierto nivel en Java, Python, C y Rust. Además de HTML, CSS y JavaScript, usados para crear esta web.

¿Qué sé hacer?

  • Además de tener un manejo demostrable de todos los lenguajes de programación ya mencionados, puedo aprender rápido a escribir en cualquier otro si es necesario.
  • Puedo instalar entornos de programación y sistemas operativos sin ningún problema.
  • Sé crear aplicaciones desde cero y hacerlas visualmente atractivas para el usuario final.
